# jharkhand-challan
Jharkhand E-Challan System is a fully digital, transparent traffic fine platform by Jharkhand Transport Dept &amp; Parivahan. Issue instant challans with live camera, photo/video proof. Auto-fetch RC/DL details. Fines: No Helmet ₹1400, No Licence ₹5000, PUC ₹1000, Triple Riding ₹3000. Pay online in 15 days or face double penalty + vehicle seizure.
